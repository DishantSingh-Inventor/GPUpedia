# GPUpedia - Graphics Card Encyclopedia

## what is this

so basically this is a website where you can see info about graphics cards. there are 10 gpus on the homepage and if you click on any of them it opens a separate page with all the specs and details.

the homepage shows basic stuff like vram, tdp, price. and the separate pages have the nerd stuff like die size, transistor count, memory bandwidth, tflops and all that.

i also added filter buttons so you can filter by nvidia amd or intel and a hamburger menu for mobile.

---

## why i made this

honestly i just wanted to build something bigger than a single page project. i have made small stuff before but never something with multiple pages all linked together properly.

also i like pc hardware so gpu felt like a good topic. i already knew some stuff about these cards so it was easier to understand what i was writing.

---

## what i used

just html css and javascript. no frameworks or anything like that.

html for all the pages, css for styling and making it look dark and clean, js for the menu and filter buttons.

i also took help from AI for some parts mainly the javascript and the spec data because researching 10 gpus manually would have taken forever

---

## what was hard

javascript was annoying. the hamburger menu kept not closing when you click a link and i had to figure out why. also the filter buttons took some time to get working properly.

css on mobile was also a pain. things would look fine on pc and then completely break on phone. had to fix that multiple times.

and managing 11 html pages was more boring than i expected. copy pasting structure across pages and making sure nothing is missing or broken.

---

## what i learned

- multi page websites are not that complicated once you understand linking
- data attributes in html are actually really useful
- making something responsive is harder than it looks
- css takes way longer than you think it will
